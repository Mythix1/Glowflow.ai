# Glowflow.ai

Welcome to Glowflow.ai! This web app helps users track their fitness progress, create workout plans, and connect with others for motivation and support.

## Setup Instructions

1. Clone the repository:
   ```bash
   git clone https://github.com/Mythix1/Glowflow.ai.git
   cd Glowflow.ai
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm start
   ```

Enjoy your fitness journey with Glowflow.ai!
